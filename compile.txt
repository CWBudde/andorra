This file will help you to get your version of Andorra compiled.

DirectX Plugin:
----------------
If you want to compile the file "AndorraDX93D.dll", you have to download the corresponding headers. You'll get them by visiting the following page:

http://www.clootie.ru/

Note: If you use newer headers, you'll also need to download the new DLL files. (d3dx9_xx.dll)
